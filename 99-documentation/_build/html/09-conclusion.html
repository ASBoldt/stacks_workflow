9. Conclusion¶

9.1. Running into problems¶

  1. Consider joining the Stacks Google group
  2. To help you solve computer related problems and start focusing back on biology, you should get this book: Practical Computing for Biologists (go for paper, because the digital edition from Sinauer is useless).
  3. Biostar is a useful bioinformatics forum.
  4. SEQanswers and SEQanswers wiki is another usefull forum for all things related to Next Generation sequencing.
  5. Of course, you can always ask Google for help.
